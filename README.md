# Kalkulators
Šis ir C++ valodā rakstīts kalkulators ar interneta kultūras jokiem.

# Būvēšana
Piezīme: Šis projekts ir paredzēts Linux (ArchLinux) operētājsistēmai.

Pārliecinies ka **build.sh** skriptam ir palaišanas tiesības ar: `chmod +x build.sh`
Vienkārši palaiž `./build.sh`.

# Palaišana
Vienkārši palaiž uzbūvēto failu `Kalkulators`.
